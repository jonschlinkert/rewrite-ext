## Usage

```js
var rewrite = require('{%= name %}');
rewrite('abc.coffee');
//=> 'abc.js'
rewrite('abc.less');
//=> 'abc.css'
```

Explicitly pass an extension to use:

```js
rewrite('./faux.js', '.foo');
//=> 'faux.foo'
```

## More examples

```js
rewrite('abc.styl');
//=> 'abc.css'
rewrite('abc.sass');
//=> 'abc.css'
rewrite('abc.scss');
//=> 'abc.css'
rewrite('abc.swig');
//=> 'abc.html'
rewrite('abc.hbs');
//=> 'abc.html'
rewrite('abc.md');
//=> 'abc.html'
rewrite('abc.tmpl');
//=> 'abc.html'
rewrite('./faux.js', '.bar');
//=> 'faux.bar'
rewrite('faux.js');
//=> 'faux.js'
rewrite('faux.css');
//=> 'faux.css'
rewrite('faux.html');
//=> 'faux.html'
```
